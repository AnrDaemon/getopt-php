These optional features add more power to Getopt in exchange for some additional configuration work.

One thing all of these have in common is that they make use of chainable methods. That makes it easy to add any number of them to an Option object, for example: {% highlight php startinline %} $getopt = new Getopt(array( new Option('o', 'option', Getopt::REQUIRED_ARGUMENT) ->setDescription('Description of option') ->setArgument(new Argument('Default value')) )); {% endhighlight %}
