You can assign a default value to any option that can take an argument. Then if that option is not given at the command line, it will assume the default value instead of null.

{% highlight php startinline %} $getopt = new Getopt(array( new Option('n', null, Getopt::REQUIRED_ARGUMENT) ->setArgument(new Argument(10)) )); {% endhighlight %}

Note: The Argument class also provides other functionality, such as argument validation.
