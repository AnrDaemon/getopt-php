Getopt can automatically generate a human-readable usage information from the specified options. Use it for quickly setting up help texts:

{% highlight php startinline %} echo $getopt->getHelpText(); {% endhighlight %}

The result will look similar to this:

{% highlight text %} Usage: my-program.php [options] [operands] Options: -a, --alpha Short and long options with no argument --beta [] Long option only with an optional argument -c Short option only with a mandatory argument {% endhighlight %}

Notes:

  • The name of the program is determined automatically.
  • The description text following each option only appears if you have set it (see here).
  • You can change the padding of the description texts by passing the desired number to getHelpText(). The default is 25.

Customizing the Message

You can change the first line of the help text to whatever you want like this:

{% highlight php startinline %} $getopt->setBanner("My custom help message for %s\n"); {% endhighlight %} The (optional) placeholder %s will be replaced by the program name.
