The recommended way of installing Getopt.php is to use Composer. Add it to your composer.json file like this:

{% highlight json %} { "require": { "ulrichsg/getopt-php": "{{ site.release-major-minor }}.*" } } {% endhighlight %}

Replace {{ site.release-major-minor }}.* by the release number you want to use (a list of releases is available on Packagist).

If not using Composer, you can download the desired release from GitHub and integrate it into your application manually. Getopt.php does not have any external dependencies.
